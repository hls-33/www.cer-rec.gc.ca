Code for Canada

La Régie de l'énergie du Canada fait équipe avec Code for Canada afin de profiter de l’aide de trois professionnels pour rendre plus accessibles aux Canadiens les données portant sur d’importants projets énergétiques et ainsi permettre aux scientifiques, propriétaires fonciers et communautés autochtones de participer pleinement au dialogue national sur l’énergie.

Consultez le site Web de Code for Canada pour en apprendre davantage à ce sujet et offrir vos services.

Découvrez ce que nous avons à offrir et comment présenter votre candidature pour avoir la chance d’entreprendre une carrière excitante dans la fonction publique fédérale. Vous trouverez ci-dessous quelques conseils pour vous aider lors du processus de candidature.

Voici une liste des Conditions pour la dotation en personnel dont vous devriez prendre connaissance afin de faciliter votre processus de candidature.

Votre candidature

Vos droits linguistiques

  • Votre candidature peut être présentée dans l’une ou l’autre des langues officielles. En outre, vous pouvez choisir la langue officielle dans laquelle se dérouleront les évaluations ou les entrevues. Veuillez toutefois noter que si vous choisissez de passer les examens et entrevues en français pour un poste pour lequel l’anglais est essentiel, vos compétences en anglais seront également évaluées.

Lettre de présentation et curriculum vitæ

  • Votre curriculum vitae et votre lettre de présentation doivent clairement énoncer que vous possédez TOUTES les qualités essentielles concernant les aptitudes linguistiques, les études et l’expérience.
  • Nous vous recommandons fortement de démontrer que vous possédez aussi les qualités représentant un atout.
  • Afin d’améliorer la clarté de votre lettre de présentation, nous vous demandons d’utiliser chaque critère de sélection comme titre et de rédiger un ou deux paragraphes précisant de quelle façon vous répondez à ces exigences.
  • Nous vous suggérons d’inclure les cours que vous avez suivis, les associations dont vous êtes membre, les prix que vous avez reçus, vos compétences particulières, les articles que vous avez publiés, un énoncé sur vos objectifs de carrière, etc. (selon l'emploi postulé).
  • Exemple de lettre de présentation et exemple de curriculum vitæ

Le processus de sélection

Lors du processus de présélection, nous examinons tous les documents que vous avez envoyés afin de vérifier si vous répondez à qualités essentielles concernant les aptitudes linguistiques, les études et l’expérience.

Voici quelques-uns des documents communément utilisés pour la présélection (liste non exhaustive) :

  • Curriculum vitæ
  • Lettre de présentation
  • Courte rédaction

Évaluation

Voici une liste des outils d’évaluation généralement utilisés par la Régie. Selon le poste, l’évaluation peut se faire de différentes façons, notamment : tests normalisés, présentation orale, scénarios de jeu de rôles et évaluation du rendement.

Examen écrit

  • Il se peut que nous vous demandions de passer un examen écrit envoyé par courriel et de le rendre dans les délais impartis (p. ex. en deux heures).
  • Il se peut que nous vous demandions de passer un examen écrit à l’interne, avant ou après une entrevue, et de le terminer dans les délais impartis (p. ex. en deux heures).

Entrevue

  • Votre candidature sera évaluée par un jury de sélection composé d’au moins deux personnes selon les critères de sélection énumérés dans l'énoncé des critères de mérite.
  • N’oubliez pas d’amener avec vous vos preuves de scolarité (diplôme, relevés de notes), une preuve de citoyenneté (acte de naissance, passeport) et votre preuve de résidence (permis de conduire) lors de votre entrevue.
  • Nous vous conseillons également d'avoir avec vous vos références (nous suggérons trois sources).

Remarques

  • Si personne n’a communiqué avec vous pour poursuivre le processus, la Régie n'est pas tenue de vous informer de l’état de votre candidature.
